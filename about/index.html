About Jeff Thoensen

QA engineer, context-driven testing, Playwright, API, and exploratory work

I focus on QA that actually matters. I build Playwright tests where they add value, wire them into CI, and spend a lot of time on exploratory testing because that’s where real problems show up. I keep suites small on purpose and treat test code like product code — reviews, refactors, and small commits over messy bulk changes.

Day to day I use Playwright, GitHub Actions, and Allure reports. Most of my work is in JavaScript and TypeScript, but I use Python when it makes sense.
